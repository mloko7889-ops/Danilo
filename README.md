# Онлайн-меню кафе "Кафе Затишок"

## Опис
Цей проєкт — сайт-візитка локального кафе із простим онлайн-меню, сторінкою контактів та сучасним дизайном. Сайт виконаний за допомогою HTML і CSS (SCSS) із адаптивним фоном та персоналізованою навігацією.

## Структура проєкту
- **index.html** — головна сторінка із назвою кафе, навігацією та футером.
- **menu.html** — сторінка з меню кафе.
- **contact.html** — сторінка з контактною інформацією.
- **index.SCSS** — стилі для сторінок (може бути також згенерований CSS).
- **image1.jpg, image2.jpg, image3.jpg** — ілюстрації або фонові зображення для сайту.

## Можливості
- Головний заголовок посередині сторінки.
- Кнопки навігації ("Контакти", "Меню") реалізовані білим текстом, 
- Сучасний напівпрозорий фон для основних блоків.
- Адаптивний фоновий малюнок для всієї сторінки.
- Контактна інформація та розділ меню на окремих сторінках.

## Як розгорнути
1. Склонуйте репозиторій або скачайте архів із файлами.
2. Відкрийте папку у редакторі коду (наприклад, VS Code).
3. Запустіть файл `index.html` у браузері.
